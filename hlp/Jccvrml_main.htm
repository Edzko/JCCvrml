Jumbo Container Crane Simulator

The JCC Vrml Simulator is an interactive motion dynamics analysis training simulator of shipping terminal cranes. The simulator features different viewpoints and various swing and sway motion cancelling control options. It interfaces with a standard Windows Game Controller interface for moving the trolley and hoisting the container. 

Further information in this documentation covers:

  1. Container Crane Motion Dynamics
  2. Swing and Sway control for Container Cranes
